留言板管理、系统管理、订单管理等信息。

个人中心，在个人中心页面中可以通过填写原密码、新密码、确认密码等信息进行添加、修改，如图5-2所示。还可以根据需要对个人信息进行添加，修改或删除等详细操作，如图5-3所示。
